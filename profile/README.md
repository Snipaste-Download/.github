## Quick Start with Snipaste Download

1. **Get Snipaste Download** — Grab the portable `.exe` (no installation required) or the installer version.
2. **Launch** — Double-click `Snipaste_Download.exe`. The main window hides to the system tray in under 1 second.
3. **Take your first capture** — Press `F1` (or your custom hotkey). Select any area of your screen.
4. **Annotate instantly** — Use the toolbar to add arrows, text, blur, shapes, or highlights in **Snipaste Download**.
5. **Pin to screen** — Press `F3` (or click the pin icon). Your capture floats above all other windows.
6. **Adjust (optional)** — Change hotkeys, default save formats, or image quality. Most people use **Snipaste Download** defaults.
7. **Work normally** — **Snipaste Download** activates instantly whenever you need to capture, annotate, or pin anything.

## Understanding Snipaste Download Core Components

**Snipaste Download** is not just another screenshot utility. It combines several essential functions:

- **Smart capture engine** — Automatic window detection, pixel-perfect area selection, and multi-monitor support.
- **Annotation toolkit** — Arrows, rectangles, ellipses, lines, text, blur, highlight, and freehand drawing.
- **Pinning system** — Turn any captured image into a floating window that stays above all applications.
- **Clipboard integration** — Copy screenshots directly to clipboard or save to disk — your choice.
- **Color picker** — Grab any pixel's color value (RGB, HEX, or HSL) with zoom and preview.
- **History manager** — Access your last 30 captures, annotations, and pins instantly.
- **Portable mode** — Run **Snipaste Download** from a USB drive on any Windows PC without admin rights.

All these work together seamlessly in **Snipaste Download**. You never need to switch between separate applications.
